# Bargain Hunt Simulator (Starter Repo)

A small starter project that simulates a *Bargain Hunt*-style episode with:
- 2 teams
- Expert-guided shopping (3 team purchases)
- Expert leftover purchase (1 extra item from remaining stall inventory)
- Appraisal + auction
- Golden Gavel rule: **all 3 team-bought items** must make a profit (both teams can qualify)

## Run
1. Create a venv (optional)
2. Install pygame:
   - `pip install pygame`
3. Run:
   - `python main.py`

## Controls
- Press **SPACE** to skip phases faster.
